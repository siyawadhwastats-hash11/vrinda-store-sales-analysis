# Vrinda Store Sales Analysis Dashboard

## Project Overview
This project analyzes the annual sales performance of Vrinda Store for the year 2022 using Microsoft Excel. An interactive dashboard was created to identify sales trends, customer behavior, order status, and sales channel performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

## Key Insights
- Women contributed the majority of total sales.
- Maharashtra was the top-performing state.
- Delivered orders accounted for the highest share of orders.
- Amazon, Myntra, and Ajio were major sales channels.
- Adult women formed the largest customer segment.
